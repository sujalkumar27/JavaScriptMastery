//innerText //It is used to get the text of an element as well as the text of all its child elements. //It returns the visible text contained in a node, excluding any HTML tags or hidden elements. //textContent //It is used to get or set the text content of an element and all its descendants. //It returns the text content of a node and its descendants, including hidden elements, but excluding any HTML tags. //innerHTML //It is used to get or set the HTML content of an element. //It returns the HTML markup contained within an element, including any child elements and their attributes. //tabular way to understand this is //Property | Description | Example //--------------|--------------------------------------------------|----------------------------------------------- //innerText | Gets/sets visible text, excluding HTML tags | element.innerText //textContent | Gets/sets all text, including hidden elements | element.textContent //innerHTML | Gets/sets HTML content, including tags | element.innerHTML
